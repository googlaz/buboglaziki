# Telegram Bot Control for Buboglaziki

Python-бот для удалённого управления проектом через Telegram.

## Возможности

| Команда | Описание |
|---------|----------|
| `/code [текст]` | Запускает `opencode-ai "[текст]"` и присылает лог |
| `/cmd [команда]` | Выполняет любую shell-команду (например, `git push`) |
| `/send [путь]` | Отправляет указанный файл как документ |
| `/start` | Приветствие Коди |
| `/help` | Список команд |
| **Любое сообщение** | Кодя отвечает через нейросеть (OpenRouter) |

**Автоматическая загрузка:** бот следит за проектом и автоматически пересылает вам:
- Все новые `.apk` файлы (где бы они ни появились)
- Любые файлы, созданные в папке `output/`

## Установка

1. **Python 3.10+** должен быть установлен.

2. **Создайте бота** через [@BotFather](https://t.me/BotFather) в Telegram, получите токен.

3. **Узнайте свой Chat ID** — напишите боту [@userinfobot](https://t.me/userinfobot), он покажет ваш числовой ID.

4. **Получите ключ OpenRouter** — зарегистрируйтесь на [openrouter.ai](https://openrouter.ai), создайте API key.

5. **Настройте `.env`:**
   ```
   BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
   MY_CHAT_ID=987654321
   OPENROUTER_API_KEY=sk-or-v1-xxxxxxxxxxxxxxxx
   ```

6. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

7. **Запустите:**
   ```bash
   python bot_control.py
   ```

## Автозагрузка Windows

### Способ 1: Планировщик задач (рекомендуется)

1. Откройте **Планировщик задач** (`taskschd.msc`)
2. **Создать задачу** → вкладка «Общие»:
   - Имя: `Buboglaziki Bot Control`
   - ✅ «Выполнять вне зависимости от регистрации пользователя»
   - ✅ «Выполнять с наивысшими правами»
3. Вкладка «Триггеры» → **Создать** → «При входе в систему»
4. Вкладка «Действия» → **Создать**:
   - Программа: `python` (или полный путь, например `C:\Python312\python.exe`)
   - Аргументы: `bot_control.py`
   - Начальная папка: `C:\buboglaziki`
5. Вкладка «Условия» → снимите ✅ «Запускать только при питании от сети»
6. OK → введите пароль пользователя

### Способ 2: Папка автозагрузки

1. Нажмите `Win + R`, введите `shell:startup`, Enter
2. Создайте файл `bot_control.bat`:
   ```bat
   @echo off
   cd /d C:\buboglaziki
   python bot_control.py
   ```
3. Положите `.bat` файл в папку автозагрузки

## Безопасность

- Бот отвечает **только** на сообщения от вашего Chat ID
- Токен и ключи хранятся в `.env` (файл в `.gitignore`)
- Никогда не коммитьте `.env` в репозиторий
